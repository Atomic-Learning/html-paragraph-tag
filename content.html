The paragraph tag is used to define a paragraph in HTML. This tag is represented by the <p> element. The opening and closing tags are used to enclose the content of the paragraph. Blanks lines and indentation within the tags in the <p> element do not affect the rendering of the paragraph. Instead, the vertical spacing and any indentation associated with the rendered paragraph is determined by the CSS properties applied to the paragraph element. A typical paragraph in a html code may look like this:

<p>
    This is a paragraph of text. It can contain multiple sentences and will be displayed as a block of text on the webpage.
</p>
